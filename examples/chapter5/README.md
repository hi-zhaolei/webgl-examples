## 着色器之前的数据传输

![webgl绘制过程](../../pic/webgl_process.png)

在顶点着色器和片元着色器之间有两个过程: **图形装配**和**光栅化**

**图形装配**将鼓励的顶点坐标装配成几何图形，也称**图元装配**，**光栅化**将装配好的几何图形转换为片元
